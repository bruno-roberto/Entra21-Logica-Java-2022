# Estrutura de repetição

## While e Do-While

- Aula sobre estruturas de repetição
- Utilização do FOR
- Utilização do While
- Utilização do Do-While
----------------------------------------------


> Existem algumas formas de descrever laços/ estruturas de repetição em Java, e uma delas e a repetição baseada em decisões

| Tipos de Repetição  | Right columns |
| ------------- |:-------------:|
| While      | so entra na repetição caso a condição de saída não seja atendida     |
| Do-While     | permite a entrada no loop antes de validar a condição de saída     |
| Função recursiva       | função que chama a propria função e com isso executa o mesmo código     |
