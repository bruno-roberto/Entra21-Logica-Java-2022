# Aprendendo matrizes

> Matrizes são vetores de vetores
> Exemplos
> - quero armazenar 3 notas para cada um dos meus 5 alunos
> - float notasAlunos \[\]\[\]= new float\[5\]\[3\];

>    - [1] ==> { 0f , 0f ,0f} 
>    - [2] ==> { 0f , 0f ,0f} 
>    - [3] ==> { 0f , 0f ,0f} 
>    - [4] ==> { 0f , 0f ,0f}
